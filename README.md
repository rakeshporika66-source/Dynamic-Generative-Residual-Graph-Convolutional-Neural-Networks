Dynamic Generative Residual Graph Convolutional Neural Networks for Electricity Theft Detection
> Developed an AI-based system to detect electricity theft by analyzing smart meter consumption data using Dynamic Generative Residual Graph Convolutional Networks (DGR-GCN).
> Preprocessed and analyzed electricity usage data with Python, Pandas, and NumPy, and modeled consumer relationships using Graph Neural Networks (GCNs).
> Built and trained the deep learning model using TensorFlow/PyTorch to identify abnormal electricity consumption patterns with high accuracy.
> Evaluated the model's performance using Scikit-learn metrics such as Accuracy, Precision, Recall, and F1-Score, and visualized the results with Matplotlib.

Technologies: Python, TensorFlow/PyTorch, Graph Convolutional Networks (GCN), Pandas, NumPy, Scikit-learn, Matplotlib, Deep Learning, Machine Learning.
